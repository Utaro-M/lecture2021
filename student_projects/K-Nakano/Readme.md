# カメラでボールとゴールを認識するサラマンダーロボット

カメラ画像から，色のついたボールとゴールを認識する

# DEMO



# Features

* ボールをキャッチするアーム搭載
* カメラが回転するようになっていて，ボールを採ったら，前方と後方を切り替えて動作する．
